# padimac

Small IMAC drum pad to have fun !
Made by Mathilde DUMOULIN & Enguerrand DE SMET

## NPM commands
### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
